# Petite Dumpling
 
This repository powers the official website for **Petite Dumpling**, a Cantonese dim sum and small plates restaurant located in Brooklyn, NY. The site showcases the menu, brand identity, and online ordering options.
 
---
 
## 🚀 Overview
 
This website serves as the official online presence for Petite Dumpling, a Brooklyn-based restaurant serving handcrafted dim sum, dumplings, and small plates all day. The site highlights the menu, chef spotlight, ordering options, and contact information.
 
---
 
## 🛠 Tech Stack
 
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Hosted on Vercel
 
---
 
## 📂 Project Structure
 
- `src/app` → App Router (homepage, layout, globals)
- `src/app/pages` → Site pages (About, Contact, etc.)
- `src/lib/components` → Shared UI components (Navbar/Footer)
- `public` → Static assets
